Class Queue<T>

A generic FIFO queue data structure.

Notes

This queue is implemented internally using a singly linked list and can grow to arbitrary size.

Example

let q = new Queue<number>([0, 1, 2]);
q.size;      // 3
q.empty;     // false
q.pop();     // 0
q.pop();     // 1
q.push(42);  // undefined
q.size;      // 2
q.pop();     // 2
q.pop();     // 42
q.pop();     // undefined
q.size;      // 0
q.empty;     // true
